# Media Extractor (M4 Pro 原生降维剥壳机)

**定位**：专为流媒体长视频 (YouTube, Bilibili, TikTok/抖音, 会议录播等) 打造的极速纯净内容提取管道。
**特性**：基于 `yt-dlp` 极速单吸音轨，并利用 Apple M4 Pro 级算力与 MLX 框架进行本地零开销的 Whisper 光速听写并产出 Markdown 文稿。

---

## Agent 使用准则 (Skill Usage)

作为 OpenClaw 生态链中的 Agent，当您面临以下场景时，请调用此技能：
1. 原线任务遇到了目标网站含有大量视频且没有文本，你无法执行提取。
2. 任务指派你去提取抖音或者油管视频讲了什么干货。

### 调取协议 (Invocation Protocol)

这并不是一个让你跟在聊天框口若悬河聊天的知识库。这是你需要用到的一把**终端改锥**。请在您的 Runtime 里开启 Terminal 工具并执行如下管道命令：

```bash
python ~/.openclaw/skills/media-extractor/scripts/extract.py <目标_流媒体_URL_或者_本地视频音频路径>
```

### 挂载参数说明：
- 输入：一个原始的在线 URL（不要带杂七杂八的数据，纯 `https://...` 即可）。
- 输出：该脚本会自动下载并执行逆向翻译，最终会在命令行返回这篇长达几十页的 `.md` 听写文本的路径。或者是截取文稿。你随后需要使用文件读取能力提取其精要！

---

## ⚠️ 防错铁律
1. **不要硬扒原 HTML**：遇到 `youtube` 或 `douyin` 的视频链接，不要固执己见地使用普通的爬虫工具！你会连一根文字的毛都爬不到。第一时间启动本脚本套壳！
2. **算力时间预估**：在召唤脚本执行前，告诉主脑或用户：“这需要一段提取时间，取决于原视频长度（按照 M4 Pro 的恐怖算力通常几百秒内）。”
